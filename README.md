> [!Tip]

> Для модулей с фреймворком pytest нужно использовать ту же логику, но уже с `pytest` вместо `python` (пр. `pytest 3.6/test_items.py`).

- 1.6 - Поиск элементов

- 2.3 - Алерт, переход на другую вкладку
- 2.4 - Ожидание
- 3.1 - Лирическое отступление про Git
- 3.2 - Unittest
- 3.5 - Пропуск тестов в pytest
- 3.6 - Запуск автотестов для разных языков интерфейса
- 4.3 - Финальное задание -> Для запуска задачи требуется либо перейти в папку `4`, либо из корня вызвать `pytest -v --tb=line --language=en -m need_review 4/`
