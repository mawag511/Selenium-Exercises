## В репозитории присутствуют следующие модули из курса
- 📘 3.1 - Лирическое отступление про Git
- 📘 3.6 - Запуск автотестов для разных языков интерфейса -> Для запуска задачи требуется либо перейти в папку `3.6`, либо из корня вызвать `pytest` для `3.6/test_items.py`
- 📘 4.3 - Финальное задание -> Для запуска задачи требуется либо перейти в папку `4`, либо из корня вызвать `pytest -v --tb=line --language=en -m need_review 4/`

> [!Warning]
> Если для финальной задачи на вашем ПК/ноутбуке не запускаются тесты или падают с ошибкой "No module named 'что-то там'", прошу впервую очередь перепроверить импорты и добавить/удалить точки в файлах
> (пр. вместо `from .base_page` писать `from base_page` или наоборот).
> На моей технике всё запускалось как загружено в репозитории.
